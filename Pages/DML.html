Insert Into Command

The INSERT INTO statement is used to insert new records in a table.

Syntax:

INSERT INTO table_name
VALUES (value1, value2, value3, ...);

Example (with Output):

            mysql> insert into student values (4064,'Priyanshu','BCA');
            Query OK, 1 row affected (0.20 sec)

mysql> select * from student;
+------+-----------+--------+
| roll | name      | course |
+------+-----------+--------+
| 4064 | Priyanshu | BCA    |
+------+-----------+--------+
1 row in set (0.12 sec)

Insert Into Command (Specified columns)

It is also possible to only insert data in specific columns.

Syntax:

INSERT INTO table_name (column1, column2, column3, ...)
VALUES (value1, value2, value3, ...);

Example (with Output):

mysql> insert into student (roll,name) values(4062,'Neeraj');
Query OK, 1 row affected (0.19 sec)

mysql> select * from student;
+------+-----------+--------+
| roll | name      | course |
+------+-----------+--------+
| 4064 | Priyanshu | BCA    |
| 4062 | Neeraj    | NULL   |
+------+-----------+--------+
2 rows in set (0.03 sec)

Insert Into Command (Multiple rows)

It is also possible to only insert multiple rows.

Syntax:

INSERT INTO table_name (column1, column2, column3, ...) VALUES 
(value1, value2, value3, ...),
(value1, value2, value3, ...), 
(value1, value2, value3, ...), 
...;

Example (with Output):

mysql> insert into student values
    -> (4001,'Anmol','BCA'),
    -> (4002,'Mario','BCA');
Query OK, 2 rows affected (0.10 sec)
Records: 2  Duplicates: 0  Warnings: 0

mysql> select * from student;
+------+-----------+--------+
| roll | name      | course |
+------+-----------+--------+
| 4064 | Priyanshu | BCA    |
| 4062 | Neeraj    | NULL   |
| 4001 | Anmol     | BCA    |
| 4002 | Mario     | BCA    |
+------+-----------+--------+
4 rows in set (0.00 sec)

Update Command

The UPDATE statement is used to modify the existing records in a table.

Syntax:

UPDATE table_name
SET column1 = value1, column2 = value2, ...
WHERE condition;

Example (with Output):

mysql> update student set course = "BCA" where roll = 4062;
Query OK, 1 row affected (0.14 sec)
Rows matched: 1  Changed: 1  Warnings: 0

mysql> select * from student;
+------+-----------+--------+
| roll | name      | course |
+------+-----------+--------+
| 4064 | Priyanshu | BCA    |
| 4062 | Neeraj    | BCA    |
| 4001 | Anmol     | BCA    |
| 4002 | Mario     | BCA    |
+------+-----------+--------+
4 rows in set (0.02 sec)

Update Command (All)

This will modify all the existing records in a table

Syntax:

UPDATE table_name
SET column1 = value1, column2 = value2, ...;

Example (with Output):

mysql> update student set course = "BCOM";
Query OK, 4 rows affected (0.09 sec)
Rows matched: 4  Changed: 4  Warnings: 0

mysql> select * from student;
+------+-----------+--------+
| roll | name      | course |
+------+-----------+--------+
| 4064 | Priyanshu | BCOM   |
| 4062 | Neeraj    | BCOM   |
| 4001 | Anmol     | BCOM   |
| 4002 | Mario     | BCOM   |
+------+-----------+--------+
4 rows in set (0.00 sec)

Delete Command

The DELETE statement is used to delete existing records in a table.

Syntax:

DELETE FROM table_name WHERE condition;

Example (with Output):

mysql> delete from student where name = 'Anmol';
Query OK, 1 row affected (0.12 sec)

mysql> select * from student;
+------+-----------+--------+
| roll | name      | course |
+------+-----------+--------+
| 4064 | Priyanshu | BCOM   |
| 4062 | Neeraj    | BCOM   |
| 4002 | Mario     | BCOM   |
+------+-----------+--------+
3 rows in set (0.00 sec)

Delete Command (All)

It is possible to delete all rows in a table without deleting the table.

Syntax:

DELETE FROM table_name;

Example (with Output):

mysql> select * from student;
+------+-----------+--------+
| roll | name      | course |
+------+-----------+--------+
| 4064 | Priyanshu | BCOM   |
| 4062 | Neeraj    | BCOM   |
| 4002 | Mario     | BCOM   |
+------+-----------+--------+
3 rows in set (0.00 sec)

mysql> delete from student;
Query OK, 3 rows affected (0.44 sec)

mysql> select * from student;
Empty set (0.00 sec)
