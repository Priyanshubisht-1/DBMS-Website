DCL Commands Grant Command Revoke Command

Grant Command

The GRANT statement is used to give user's permissions to a database

Syntax:

GRANT privilege_name
ON object_name
TO {user_name |PUBLIC |role_name}
[WITH GRANT OPTION];

Example (with Output):

mysql> create user tester identified by 'tester123';
Query OK, 0 rows affected (1.31 sec)

mysql> grant select,insert on student to tester;
Query OK, 0 rows affected (0.12 sec)

Revoke Command

The REVOKE statement is used to take back permissions from a user

Syntax:

REVOKE privilege_name
ON object_name
FROM {user_name |PUBLIC |role_name}

Example (with Output):

mysql> create user tester identified by 'tester123';
Query OK, 0 rows affected (1.31 sec)

mysql> grant select,insert on student to tester;
Query OK, 0 rows affected (0.12 sec)

mysql> revoke select,insert on student from tester;
Query OK, 0 rows affected (0.11 sec)
