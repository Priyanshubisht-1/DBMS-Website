Inner Join

The INNER JOIN keyword selects records that have matching values in both tables.

Syntax:

SELECT column_name(s)
FROM table1
INNER JOIN table2
ON table1.column_name = table2.column_name;

Example (with Output):

mysql> select * from student;
+--------+-----------+------------+
| rollno | name      | contact    |
+--------+-----------+------------+
|      1 | Priyanshu | 1234567890 |
|      2 | Neeraj    | 1234567890 |
|      3 | Akhil     | 1234567890 |
+--------+-----------+------------+
3 rows in set (0.00 sec)

mysql> select * from marks;
+--------+------+------+------+
| rollno | sub1 | sub2 | sub3 |
+--------+------+------+------+
|      1 |   45 |   65 |   78 |
|      3 |   55 |   81 |   97 |
|      4 |   12 |    1 |    7 |
+--------+------+------+------+
3 rows in set (0.00 sec)

mysql> select name,sub1+sub2+sub3 'Total' from student 
    -> inner join marks on student.rollno = marks.rollno;
+-----------+-------+
| name      | Total |
+-----------+-------+
| Priyanshu |   188 |
| Akhil     |   233 |
+-----------+-------+
2 rows in set (0.00 sec)

Left Join (Left Outer Join)

The LEFT JOIN keyword returns all records from the left table (table1), and the matching records from the right table (table2). The result is 0 records from the right side, if there is no match.

Syntax:

SELECT column_name(s)
FROM table1
LEFT JOIN table2
ON table1.column_name = table2.column_name;

Example (with Output):

mysql> select * from student;
+--------+-----------+------------+
| rollno | name      | contact    |
+--------+-----------+------------+
|      1 | Priyanshu | 1234567890 |
|      2 | Neeraj    | 1234567890 |
|      3 | Akhil     | 1234567890 |
+--------+-----------+------------+
3 rows in set (0.00 sec)

mysql> select * from marks;
+--------+------+------+------+
| rollno | sub1 | sub2 | sub3 |
+--------+------+------+------+
|      1 |   45 |   65 |   78 |
|      3 |   55 |   81 |   97 |
|      4 |   12 |    1 |    7 |
+--------+------+------+------+
3 rows in set (0.00 sec)

mysql> select name,sub1+sub2+sub3 'Total' from student 
    -> left join marks on student.rollno = marks.rollno;
+-----------+-------+
| name      | Total |
+-----------+-------+
| Priyanshu |   188 |
| Neeraj    |  NULL |
| Akhil     |   233 |
+-----------+-------+
3 rows in set (0.00 sec)

Right Join (Right Outer Join)

ON table1.column_name = table2.column_name;

Full join (Full Outer Join)

The FULL OUTER JOIN keyword returns all records when there is a match in left (table1) or right (table2) table records.

Note : There is no direct Full Outer Join keyword instead need to use Union of left outer join and right outer join

Example (with Output):

mysql> select * from student;
+--------+-----------+------------+
| rollno | name      | contact    |
+--------+-----------+------------+
|      1 | Priyanshu | 1234567890 |
|      2 | Neeraj    | 1234567890 |
|      3 | Akhil     | 1234567890 |
+--------+-----------+------------+
3 rows in set (0.00 sec)

mysql> select * from marks;
+--------+------+------+------+
| rollno | sub1 | sub2 | sub3 |
+--------+------+------+------+
|      1 |   45 |   65 |   78 |
|      3 |   55 |   81 |   97 |
|      4 |   12 |    1 |    7 |
+--------+------+------+------+
3 rows in set (0.00 sec)

mysql> select name,sub1+sub2+sub3 'Total' from student left outer join 
    -> marks on student.rollno = marks.rollno union select name
    -> ,sub1+sub2+sub3 'Total' from student right join marks 
    -> on student.rollno = marks.rollno;
+-----------+-------+
| name      | Total |
+-----------+-------+
| Priyanshu |   188 |
| Neeraj    |  NULL |
| Akhil     |   233 |
| NULL      |    20 |
+-----------+-------+
4 rows in set (0.19 sec)

Cross join

The CROSS JOIN keyword returns all records from both tables (table1 and table2)

Syntax:

SELECT column_name(s)
FROM table1
CROSS JOIN table2;

Example (with Output):

mysql> select * from student;
+--------+-----------+------------+
| rollno | name      | contact    |
+--------+-----------+------------+
|      1 | Priyanshu | 1234567890 |
|      2 | Neeraj    | 1234567890 |
|      3 | Akhil     | 1234567890 |
+--------+-----------+------------+
3 rows in set (0.00 sec)

mysql> select * from marks;
+--------+------+------+------+
| rollno | sub1 | sub2 | sub3 |
+--------+------+------+------+
|      1 |   45 |   65 |   78 |
|      3 |   55 |   81 |   97 |
|      4 |   12 |    1 |    7 |
+--------+------+------+------+
3 rows in set (0.00 sec)

mysql> select name,sub1+sub2+sub3 'Total' from student cross join marks;

| name      | Total |
+-----------+-------+

| Neeraj    |   188 |
| Priyanshu |   188 |

Self Join

A SELF JOIN is a regular join, but the table is joined with itself.

Note :T1 and T2 are different table aliases for the same table.

Syntax:

SELECT column_name(s)
FROM table1 T1, table1 T2
WHERE condition;

Example (with Output):

mysql> select * from student;
+--------+-----------+------------+
| rollno | name      | contact    |
+--------+-----------+------------+
|      1 | Priyanshu | 1234567890 |
|      2 | Neeraj    | 1234567890 |
|      3 | Akhil     | 1234567890 |
+--------+-----------+------------+
3 rows in set (0.00 sec)

mysql> select * from marks;
+--------+------+------+------+
| rollno | sub1 | sub2 | sub3 |
+--------+------+------+------+
|      1 |   45 |   65 |   78 |
|      3 |   55 |   81 |   97 |
|      4 |   12 |    1 |    7 |
+--------+------+------+------+
3 rows in set (0.00 sec)

mysql> select s1.name,s2.name,s1.contact from student s1
    -> , student s2 where s1.contact = s2.contact;
+-----------+-----------+------------+
| name      | name      | contact    |
+-----------+-----------+------------+
| Akhil     | Priyanshu | 1234567890 |
| Neeraj    | Priyanshu | 1234567890 |
| Priyanshu | Priyanshu | 1234567890 |
| Akhil     | Neeraj    | 1234567890 |
| Neeraj    | Neeraj    | 1234567890 |
| Priyanshu | Neeraj    | 1234567890 |
| Akhil     | Akhil     | 1234567890 |
| Neeraj    | Akhil     | 1234567890 |
| Priyanshu | Akhil     | 1234567890 |

9 rows in set (0.02 sec)

Union Operator

A Union operator is used to combine the result-set of two or more SELECT statements

  • Every SELECT statement within UNION must have the same number of columns

  • The columns must also have similar data types

  • The columns in every SELECT statement must also be in the same order

Syntax:

SELECT column_name(s) FROM table1
UNION
SELECT column_name(s) FROM table2;

Example (with Output):

mysql> select * from student;
+--------+-----------+------------+
| rollno | name      | contact    |
+--------+-----------+------------+
|      1 | Priyanshu | 1234567890 |
|      2 | Neeraj    | 1234567890 |
|      4 | Ak        | 1234567890 |
+--------+-----------+------------+
3 rows in set (0.02 sec)

mysql> select * from student2;
+--------+-----------+------------+
| rollno | name      | contact    |
+--------+-----------+------------+
|      1 | Priyanshu | 1234567890 |
|      2 | Neeraj    | 1234567890 |
|      3 | Aman      | 1234567890 |
|      4 | Ak        | 1234567890 |
+--------+-----------+------------+
4 rows in set (0.00 sec)

mysql> select * from student union select * from student2;
+--------+-----------+------------+
| rollno | name      | contact    |
+--------+-----------+------------+
|      1 | Priyanshu | 1234567890 |
|      2 | Neeraj    | 1234567890 |
|      4 | Ak        | 1234567890 |
|      3 | Aman      | 1234567890 |
+--------+-----------+------------+
4 rows in set (0.00 sec)

Union All Operator

A Union operator selects only distinct values by default. To allow duplicate values, use UNION ALL

Syntax:

SELECT column_name(s) FROM table1
UNION ALL
SELECT column_name(s) FROM table2;

Example (with Output):

mysql> select * from student;

+--------+-----------+------------+

+--------+-----------+------------+
3 rows in set (0.02 sec)

mysql> select * from student2;

|      1 | Priyanshu | 1234567890 |

|      4 | Ak        | 1234567890 |

Intersect Operator

The INTERSECT operator in SQL is used to retrieve the records that are identical/common between the result sets of two or more tables.

Note : There is no such operator in MySQL instead you can use inner join

Syntax:

SELECT column_name(s)
FROM table1
INNER JOIN table2
ON table1.column_name = table2.column_name;

Example (with Output):

mysql> select * from student;
+--------+-----------+------------+

+--------+-----------+------------+
|      1 | Priyanshu | 1234567890 |
|      2 | Neeraj    | 1234567890 |
|      3 | Akhil     | 1234567890 |
+--------+-----------+------------+
3 rows in set (0.00 sec)

mysql> select * from marks;

|      3 |   55 |   81 |   97 |
|      4 |   12 |    1 |    7 |
+--------+------+------+------+
3 rows in set (0.00 sec)

mysql> select name,sub1+sub2+sub3 'Total' from student

Minus Operator

The MINUS operator is used to return all rows in the first SELECT statement that are not returned by the second SELECT statement.

Note : There is no such operator in MySQL instead you can use left join with where clause

Syntax:

SELECT column_name(s)
FROM table1
LEFT JOIN table2
ON table1.column_name = table2.column_name
WHERE table2.column_name IS NULL;

Example (with Output):

mysql> select * from student;
+--------+-----------+------------+
| rollno | name      | contact    |
+--------+-----------+------------+
|      1 | Priyanshu | 1234567890 |
|      2 | Neeraj    | 1234567890 |
|      4 | Ak        | 1234567890 |
+--------+-----------+------------+
3 rows in set (0.00 sec)

mysql> select * from marks;
+--------+------+------+------+
| rollno | sub1 | sub2 | sub3 |
+--------+------+------+------+
|      1 |   45 |   65 |   78 |
|      3 |   55 |   81 |   97 |
|      4 |   12 |    1 |    7 |
+--------+------+------+------+
3 rows in set (0.00 sec)

mysql> select name,sub1+sub2+sub3 'Total' from student
    -> left join marks on student.rollno = marks.rollno 
    -> where marks.rollno is NULL;
+--------+-------+
| name   | Total |
+--------+-------+
| Neeraj |  NULL |
+--------+-------+
1 row in set (0.00 sec)
