TCL Commands Commit Command Rollback Command

Commit Command

A COMMIT is an SQL command that enables the user to permanently save the current transactions or database statements in the relational database or the transaction table.

Since the changes are permanently saved in the database, it is impossible to go back to the previous states in which the database began before the transaction began in the system.

Note : Turn off the auto-commit by setting the value of auto-commit as 0.

mysql> SET autocommit = 0;

Syntax:

COMMIT;

Example (with Output):

mysql> select * from student;
+------+-----------+--------+
| roll | name      | course |
+------+-----------+--------+
| 4001 | Anmol     | BCA    |
| NULL | Mario     | BCOM   |
| 4062 | Neeraj    | BCA    |
| 4064 | Priyanshu | BCA    |
| NULL | Max       | BCOM   |
| 4061 | Neema     | BCOM   |
| 4031 | Krish     | BTECH  |
| 4021 | Rahul     | BTECH  |
+------+-----------+--------+
8 rows in set (0.17 sec)

mysql> delete from student;
Query OK, 8 rows affected (4.15 sec)

mysql> commit;
Query OK, 0 rows affected (0.10 sec)

mysql> rollback;
Query OK, 0 rows affected (0.00 sec)

mysql> select * from student;
Empty set (0.00 sec)

Rollback Command

The ROLLBACK command is used to get back to the previous permanent status of the table, which is saved by the commit command.

Note : if you have already committed your recent changes, you cannot rollback your transaction. In that case, you can only roll to the last permanent change.

-> Turn off the auto-commit by setting the value of auto-commit as 0.

mysql> SET autocommit = 0;

Syntax:

ROLLBACK;

Example (with Output):

mysql> select * from student;
+------+-----------+--------+
| roll | name      | course |
+------+-----------+--------+
| 4064 | Priyanshu | BCA    |
| 4062 | Neeraj    | BCA    |
+------+-----------+--------+
2 rows in set (0.02 sec)

mysql> delete from student;
Query OK, 2 rows affected (0.19 sec)

mysql> select * from student;
Empty set (0.00 sec)

mysql> rollback;
Query OK, 0 rows affected (0.05 sec)

mysql> select * from student;
+------+-----------+--------+
| roll | name      | course |
+------+-----------+--------+
| 4064 | Priyanshu | BCA    |
| 4062 | Neeraj    | BCA    |
+------+-----------+--------+
2 rows in set (0.00 sec)
