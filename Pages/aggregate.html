Count() function

The Count() function returns the number of rows (but ignore NULL values)

Syntax:

SELECT COUNT(column_name)
FROM table_name;

Example (with Output):

mysql> select * from student;
+------+-----------+--------+
| roll | name      | course |
+------+-----------+--------+
| 4001 | Anmol     | BCA    |
| NULL | Mario     | BCA    |
| 4062 | Neeraj    | BCA    |
| 4064 | Priyanshu | BCA    |
| NULL | Max       | BCA    |
+------+-----------+--------+
5 rows in set (0.00 sec)

mysql> select count(roll) from student;
+-------------+
| count(roll) |
+-------------+
|           3 |
+-------------+
1 row in set (0.00 sec)
