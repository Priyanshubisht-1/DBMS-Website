Case 1

To display the 1st, 2nd, and 3rd position holders (based on Total marks) from the student Table based on following conditions:

  • The student must have scored more than 60% marks in each subject.

  • The student must not have failed in any subject.

  • The student must not be absent for any subject's exam.

Table:

mysql> desc student_marks;

+-------+-------------+------+-----+---------+-------+

| name  | varchar(20) | YES  |     | NULL    |       |
| sub1  | varchar(5)  | YES  |     | NULL    |       |
| sub2  | varchar(5)  | YES  |     | NULL    |       |
| sub3  | varchar(5)  | YES  |     | NULL    |       |
| sub4  | varchar(5)  | YES  |     | NULL    |       |

mysql> SELECT roll, name, sub1 + sub2 + sub3 + sub4 AS Total_Marks
    -> FROM student_marks
    -> WHERE sub1 IS NOT NULL AND sub2 IS NOT NULL AND sub3 IS NOT NULL AND sub4 IS NOT NULL
