Case 1

In SQL, a view is a virtual table based on the result-set of an SQL statement.

Syntax:

CREATE VIEW view_name AS
SELECT column1, column2, ...
FROM table_name
WHERE condition;

Example (with Output):

mysql> create view view_std as select name from student;
Query OK, 0 rows affected (1.14 sec)

mysql> select * from view_std;
+-----------+
| name      |
+-----------+
| Anmol     |
| Mario     |
| Neeraj    |
| Priyanshu |
| Max       |
| Neema     |
| Krish     |
| Rahul     |
+-----------+
8 rows in set (0.17 sec)

Case 2

A view can be updated with the CREATE OR REPLACE VIEW statement.

Syntax:

CREATE OR REPLACE VIEW view_name AS
SELECT column1, column2, ...
FROM table_name
WHERE condition;

Example (with Output):

mysql> create or replace view view_std as select distinct course from student;
Query OK, 0 rows affected (0.22 sec)

mysql> select * from view_std;
+--------+
| course |
+--------+
| BCA    |
| BCOM   |
| BTECH  |
+--------+
3 rows in set (0.06 sec)
