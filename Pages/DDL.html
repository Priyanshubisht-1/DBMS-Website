Create Command

The CREATE TABLE statement is used to create a new table in a database

Syntax:

CREATE TABLE table_name (
  column1 datatype,
  column2 datatype,
  column3 datatype,
 ....
);

Example (with Output):

mysql> create table student (roll int,name varchar(20),course varchar(20));
Query OK, 0 rows affected (0.96 sec)

mysql> show tables;
+----------------+
| Tables_in_temp |
+----------------+
| student        |
+----------------+
1 row in set (0.00 sec)

Alter Command (Add)

To add a column in a table

Syntax:

ALTER TABLE table_name
ADD column_name datatype;

Example (with Output):

mysql> alter table student add contact bigint;
Query OK, 0 rows affected (0.39 sec)
Records: 0  Duplicates: 0  Warnings: 0

mysql> desc student;
+---------+-------------+------+-----+---------+-------+
| Field   | Type        | Null | Key | Default | Extra |

| name    | varchar(20) | YES  |     | NULL    |       |
| course  | varchar(20) | YES  |     | NULL    |       |
| contact | bigint      | YES  |     | NULL    |       |
+---------+-------------+------+-----+---------+-------+
4 rows in set (0.00 sec)

Alter Command (Modify)

To change the data type of a column in a table

Syntax:

ALTER TABLE table_name
MODIFY COLUMN column_name datatype;

Example (with Output):

mysql> alter table student modify contact char(10);
Query OK, 0 rows affected (1.85 sec)
Records: 0  Duplicates: 0  Warnings: 0

mysql> desc student;

| contact | char(10)    | YES  |     | NULL    |       |

4 rows in set (0.00 sec)

Alter Command (Drop)

To delete a column in a table

Syntax:

ALTER TABLE table_name
DROP COLUMN column_name;

Example (with Output):

mysql> alter table student drop course;
Query OK, 0 rows affected (2.52 sec)
Records: 0  Duplicates: 0  Warnings: 0

mysql> desc student;

Drop Command

The DROP TABLE statement is used to drop an existing table in a database.

Syntax:

DROP TABLE table_name;

Example (with Output):

mysql> drop table student;
Query OK, 0 rows affected (3.98 sec)

mysql> show tables;
+----------------+
| Tables_in_temp |
+----------------+
| students_rec   |
+----------------+
1 row in set (0.19 sec)

Truncate Command

The TRUNCATE TABLE statement is used to delete the data inside a table, but not the table itself.

Syntax:

TRUNCATE TABLE table_name;

Example (with Output):

mysql> select * from students_rec;
+------+-----------+--------+
| roll | name      | course |
+------+-----------+--------+
| 4064 | Priyanshu | BCA    |
+------+-----------+--------+
1 row in set (0.04 sec)

mysql> truncate table students_rec;
Query OK, 0 rows affected (1.74 sec)

mysql> select * from students_rec;
Empty set (0.00 sec)
