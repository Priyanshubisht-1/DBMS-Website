View Commands Create View Command Create or Replace View Command Drop View Command

Create View Command

In SQL, a view is a virtual table based on the result-set of an SQL statement.

A view contains rows and columns, just like a real table. The fields in a view are fields from one or more real tables in the database.

A view is created with the CREATE VIEW statement

Syntax:

CREATE VIEW view_name AS
SELECT column1, column2, ...
FROM table_name
WHERE condition;

Example (with Output):

mysql> create view view_std as select name from student;
Query OK, 0 rows affected (1.14 sec)

mysql> select * from view_std;
+-----------+
| name      |
+-----------+
| Anmol     |
| Mario     |
| Neeraj    |
| Priyanshu |
| Max       |
| Neema     |
| Krish     |
| Rahul     |
+-----------+
8 rows in set (0.17 sec)

Create or Replace View Command

A view can be updated with the CREATE OR REPLACE VIEW statement.

Syntax:

CREATE OR REPLACE VIEW view_name AS
SELECT column1, column2, ...
FROM table_name
WHERE condition;

Example (with Output):

mysql> create or replace view view_std as select distinct course from student;
Query OK, 0 rows affected (0.22 sec)

mysql> select * from view_std;
+--------+
| course |
+--------+
| BCA    |
| BCOM   |
| BTECH  |
+--------+
3 rows in set (0.06 sec)

Drop View Command

A view is deleted with the DROP VIEW statement

Syntax:

DROP VIEW view_name;

Example (with Output):

mysql> drop view view_std;
Query OK, 0 rows affected (0.18 sec)

mysql> select * from view_std;
ERROR 1146 (42S02): Table 'temp.view_std' doesn't exist
